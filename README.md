# CareerConnect (Static Demo)

A simple LinkedIn-style job provider app with:

- Jobs feed with filters and sample listings
- Live Classes (BBA, BCA, BTech, AIML, and more) with ₹1199 enroll
- Resume upload tool
- Resume Builder enrollment flow for ₹299

How to run

1. Open `index.html` directly in a browser, or
2. Serve the folder with any static server:

```bash
python3 -m http.server 8000 --directory /workspace
```

Then visit `http://localhost:8000/index.html`.

Notes

- All enrollments are simulated and stored in `localStorage` only.
- No backend is required; this is a front-end demo.
